# متن · Matn

**قارئ ماركداون عربي، هادئ، من اليمين إلى اليسار.**

## المشكلة

الطرفية وكثير من المحرّرات تعرض ماركداون العربي بشكل رديء: بلا خوارزمية اتجاه ثنائي،
وصل حروف مكسور، وأسطر مختلطة (عربي/لاتيني) تنهار. **متن** يصيّر ملف `.md` في المتصفح
بضبط `dir="auto"` لكل كتلة، فتنساب العربية يميناً بينما يبقى الكود والإنجليزية LTR — ضمن
تجربة قراءة مصمّمة للنصوص العربية الطويلة.

## المزايا

- 🪶 **RTL حقيقي** — اتجاه لكل كتلة؛ العربية محاذاة يمين، الكود/الإنجليزية يسار.
- 🎨 **٤ سمات قراءة** — فاتح · سيبيا · داكن · ليلي (أسود صافٍ). تتبع نظامك افتراضياً.
- 🔤 **٥ خطوط عربية مضمّنة** — النظام، نوتو نسخ، أميري، IBM Plex، تجوّل. كلها OFL و**تعمل دون اتصال**.
- 🔧 **تحكّم بالقراءة** — الخط، الحجم، تباعد الأسطر، وعرض النص — كلها محفوظة.
- 🌈 **تلوين صياغي** متكيّف مع السمة (highlight.js).
- 🧭 **فهرس محتويات** تلقائي مع تتبّع التمرير، ومراسٍ للعناوين.
- ♻️ **تحديث حيّ** — حرّر الملف بأي محرّر، فتتحدّث المعاينة عند الحفظ.
- 🖱️ **سحب وإفلات** — أفلِت أي `.md` على النافذة لقراءته.
- 🖨️ **طباعة / تصدير PDF** بنقرة، بتخطيط طباعة نظيف.
- ⏱️ **وقت القراءة وعدد الكلمات**، ✅ قوائم مهام GFM، جداول، اقتباسات.
- 📦 **بلا تبعيات تشغيل** — Node خالص وأصول مُضمَّنة.

## التثبيت

**عام (من GitHub):**
```bash
npm install -g Ajarallah/matn
matn README.md
```

**تشغيل دون تثبيت:**
```bash
npx github:Ajarallah/matn README.md
```

**من المصدر:**
```bash
git clone https://github.com/Ajarallah/matn.git
cd matn && npm link
matn README.md
```

> يتطلّب Node.js ≥ 18. بلا أي تبعيات أخرى.

## الاستعمال

```bash
matn <file.md>        # فتح ملف واحد
matn ./docs           # تصفّح مجلد (قائمة جانبية)
matn                  # فتح المجلد الحالي
matn PLAN.md -p 5000  # منفذ مخصّص
```

الخيارات: `-p, --port` · `--host` · `--no-open` · `-h, --help` · `-v, --version`.

يفتح متن المتصفح تلقائياً ويعيد استخدام نسخة قائمة، فـ`matn a.md` ثم `matn b.md` يفتحان في نافذة واحدة.

### داخل المتصفح

- **⚙** للسمة والخط والحجم والتباعد والعرض — كلها محفوظة.
- **اسحب** أي `.md` إلى النافذة لفتحه.
- **🖨️** للطباعة / تصدير PDF نظيف.
- مفاتيح: `+` / `−` الحجم · `g` / `G` أعلى / أسفل · `Esc` إغلاق اللوحة.

## فتح `.md` بالنقر المزدوج (macOS، اختياري)

اجعل متن القارئ الافتراضي لملفات ماركداون:

```bash
brew install duti
duti -s com.ajarallah.matn net.daringfireball.markdown all
```

أو لكل ملف: **Get Info ← Open with ← Change All**. راجع [`scripts/`](./scripts) لبناء تطبيق Finder جاهز.

## كيف يعمل

متن خادم HTTP محلي صغير (~٢٠٠ سطر، بلا تبعيات). يصيّر الماركداون عبر
[marked](https://github.com/markedjs/marked)، ويضع `dir="auto"` على كل كتلة نص لضبط الاتجاه،
ويلوّن الكود عبر [highlight.js](https://github.com/highlightjs/highlight.js)، ويبثّ أحداث
التحديث الحيّ عبر SSE. الخطوط والمكتبات مُضمَّنة، فيعمل دون اتصال ولا يرسل شيئاً للخارج.
