## المشكلة

الطرفية وكثير من المحرّرات تعرض ماركداون العربي بشكل رديء: بلا خوارزمية اتجاه ثنائي،
ووصل حروف مكسور، وأسطر مختلطة (عربي/لاتيني) تنهار، وعنوان يبدأ بكلمة لاتينية يقلب السطر
كله إلى اليسار. **متن** يصيّر ملف `.md` في المتصفح كمستند عربي حقيقي من اليمين إلى اليسار،
بينما يبقى الكود والإنجليزية داخله من اليسار إلى اليمين — ضمن تجربة قراءة مصمّمة للنصوص
العربية الطويلة.

## المزايا

**القراءة**

- 🪶 **RTL حقيقي** — المستند يتبع لغته *الغالبة*، فيبقى الملف العربي من اليمين إلى اليسار حتى لو بدأ عنوان أو سطر بحرف لاتيني. النصوص اللاتينية تُقرأ من اليسار إلى اليمين داخل السطر.
- 🎨 **أربع سمات** — فاتح · سيبيا · داكن · ليلي (أسود OLED)؛ تتبع نظامك افتراضيا.
- 🔤 **خطوط عربية مضمّنة** — النظام، نوتو نسخ، أميري، IBM Plex، تجوّل (كلها SIL OFL، مضمّنة، **تعمل دون اتصال**). وخط ثمانية (Display و Text) اختياري.
- 🔧 **تحكّم بالقراءة** — الخط، الحجم، تباعد الأسطر، عرض العمود، ومحاذاة النص (من اليمين / ضبط الأسطر)؛ كل اختيار محفوظ محليا.
- 🌍 **واجهة ثنائية اللغة** — بدّل الواجهة كاملة بين العربية والإنجليزية.

**المحتوى**

- 🌈 **تلوين صياغي** متكيّف مع السمة (highlight.js).
- 🧮 **معادلات** — `$…$` ضمن السطر و `$$…$$` ككتلة، عبر KaTeX ودون اتصال.
- 📊 **مخططات Mermaid** — داخل النص، متكيّفة مع السمة؛ مرّر المؤشّر فوقها للتكبير.
- 💬 **صناديق تنبيه GFM** — `> [!NOTE]`، `[!TIP]`، `[!WARNING]` … بأسلوب لكل نوع.
- 🔗 **روابط ويكي** — `[[صفحة]]` و `[[صفحة|اسم]]`، كما في أوبسيديان.
- 📝 **حواشٍ**، و**بطاقة بيانات وصفية (YAML frontmatter)**، وقوائم مهام، وجداول، واقتباسات.
- 🖼️ **تكبير بالتمرير** — كبّر المخططات والصور عند المؤشّر.

**التنقّل والملفات**

- 🧭 **فهرس محتويات** تلقائي مع تتبّع التمرير ومراسٍ للعناوين.
- 🗂️ **شجرة ملفات** — افتح مجلدا لتتصفّح شجرة أدلة متداخلة قابلة للطي.
- 🔎 **بحث داخل المستند** — اضغط `/` للبحث والتنقّل بين النتائج.
- ♻️ **تحديث حيّ** — حرّر بأي محرّر، فتتحدّث المعاينة عند الحفظ.
- 🖱️ **سحب وإفلات** لأي `.md` على النافذة.

**الإخراج**

- 📤 **تصدير** — PDF، أو HTML مستقل، أو Word‏ (`.docx`)، أو EPUB 3 (تقدّم صفحات RTL)، أو ماركداون خام.
- 🖨️ **طباعة** بتخطيط نظيف.

**الأساسات**

- 📦 **بلا تبعيات تشغيل** — Node خالص وأصول مضمّنة، يعمل دون اتصال ولا يرسل شيئا للخارج.
- 🔒 **محصور** — يرتبط بـ `127.0.0.1`، ويخدم فقط من المجلد الذي فتحته، ويهرّب HTML الخام، ويمنع مخططات الروابط غير الآمنة.

## التثبيت

**عام (من GitHub):**
```bash
npm install -g Ajarallah/matn
matn README.md
```

**تشغيل دون تثبيت:**
```bash
npx github:Ajarallah/matn README.md
```

**من المصدر:**
```bash
git clone https://github.com/Ajarallah/matn.git
cd matn && npm link
matn README.md
```

> يتطلّب Node.js ≥ 18. بلا أي تبعيات أخرى.

## الاستعمال

```bash
matn <file.md>        # فتح ملف واحد
matn ./docs           # تصفّح مجلد (شجرة ملفات جانبية)
matn                  # فتح المجلد الحالي
matn PLAN.md -p 5000  # منفذ مخصّص
```

الخيارات: `-p, --port` · `--host` · `--no-open` · `-h, --help` · `-v, --version`.

يفتح متن المتصفح تلقائيا ويعيد استخدام نسخة قائمة، فـ`matn a.md` ثم `matn b.md` يفتحان في نافذة واحدة.

### داخل المتصفح

- **⚙** للّغة والسمة والخط والحجم والتباعد والعرض والمحاذاة — كلها محفوظة.
- **اسحب** أي `.md` إلى النافذة لفتحه.
- **حفظ ▾** يصدّر PDF / HTML / Word / EPUB / ماركداون؛ و**🖨️** للطباعة.
- اضغط **/** للبحث. مفاتيح: `+` / `−` الحجم · `g` / `G` أعلى / أسفل · `Esc` إغلاق اللوحة.

## فتح `.md` بالنقر المزدوج

**macOS** — اجعل متن القارئ الافتراضي لماركداون بخطوة واحدة:

```bash
bash scripts/install-macos.sh --default
```

يبني تطبيق Finder صغيرا ويسجّله لملفات `.md`. لتوجيه تثبيت قائم يدويا:
**Get Info ← Open with ← Change All**.

**Linux** — `bash scripts/install-linux.sh --default` يضيف مدخل `.desktop` ويجعل متن معالِج `text/markdown`.
**Windows** — الأمر `matn file.md` يعمل من أي طرفية؛ واربط `.md` عبر
*Open with ← Choose another app* مشيرا إلى `matn`.

## كيف يعمل

متن خادم HTTP محلي صغير (`src/server.mjs`، بلا تبعيات). يصيّر الماركداون عبر
[marked](https://github.com/markedjs/marked)، ويمنح كل كتلة نص اتجاه المستند الغالب لضبط
الاتجاه الثنائي، ويلوّن الكود عبر [highlight.js](https://github.com/highlightjs/highlight.js)،
ويصيّر المعادلات عبر [KaTeX](https://katex.org) والمخططات عبر [Mermaid](https://mermaid.js.org)
(كلاهما بتحميل كسول)، ويبثّ التحديث الحيّ عبر SSE. الخطوط والمكتبات مضمّنة، فيعمل دون اتصال
ولا يرسل شيئا للخارج.

## الأمان

يرتبط متن افتراضيا بـ `127.0.0.1`، ولا يخدم ملفات ماركداون والصور النقطية المشار إليها
إلا من الملف أو المجلد الذي فتحته. يهرّب HTML الخام داخل الماركداون، ويمنع مخططات الروابط
غير الآمنة مثل `javascript:`. تجنّب `--host 0.0.0.0` إلا إذا كنت تريد عمدا إتاحة القارئ
لأجهزة أخرى على شبكتك. راجع [SECURITY.md](./SECURITY.md).
